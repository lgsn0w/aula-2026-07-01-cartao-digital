# Cartão Digital — Lucas Gabriel

Página pessoal feita em HTML e CSS puro, sem frameworks. Funciona como um cartão de visita digital: apresentação, habilidades, projetos e links.

## O que tem aqui

- Header com foto, nome e descrição
- Seções de habilidades, projetos e filmes favoritos
- Link pro GitHub
- Layout responsivo com cards e efeitos visuais simples

## Tecnologias

- HTML5
- CSS3 (Flexbox, gradientes, border-radius, box-shadow)

## Sobre o projeto

Feito durante a aula de HTML/CSS do dia 01/07/2026. O objetivo foi praticar estruturação semântica e estilização com CSS, saindo de exemplos básicos e chegando num resultado que dá pra usar de verdade.

A página tem um avatar circular, tags de habilidades em formato de pílula, cards com borda lateral colorida e um fundo gradiente no cabeçalho.

## Como rodar

Abre o `index.html` no navegador. Sem dependências, sem build, sem nada.
